Setup of Fruit in ShredderClassic

1. Choose in the Menu "Extras" > "Engines" and then "Install Engine":

2. In the following Window select "UCI-Engine" and click "OK":

3. In the next window click on the three dots next to the Input field and select the Fruit-Executable (by default in the folder C:\Program Files\Fruit):

4. Open it and it's done! You can now select Fruit 2.2 in the Engine Management.
